Analysis Modules

PyFlow provides a comprehensive suite of static analysis modules for understanding and analyzing Python programs. This section documents each analysis module and its capabilities.

Overview

PyFlow’s analysis modules are organized into several categories:

IR Construction

  • Control Flow Graph (CFG) Analysis - Control Flow Graph construction and analysis

  • Control Dependence Graph (CDG) - Control Dependence Graph analysis

  • Control Flow Graph Intermediate Representation (CFG-IR) - Control Flow Graph Intermediate Representation

  • Call Graph Analysis - Call Graph construction and analysis

Data Flow Analysis

  • dataflowIR - Data Flow Intermediate Representation

  • fsdf - Flow-Sensitive Data Flow analysis

  • Inter-procedural Analysis (IPA) - Inter-procedural Analysis

  • Constraint-based Analysis (CPA) - Constraint-based Analysis

Shape and Type Analysis

  • Shape Analysis - Shape analysis for data structures

  • Store Graph Analysis - Store graph analysis for object relationships

Specialized Analysis

  • numbering - Program point numbering

  • Variable Lifetime Analysis - Variable lifetime analysis

  • dump - Analysis result dumping and visualization
